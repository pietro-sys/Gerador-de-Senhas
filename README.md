# Gerador de Senhas 🔐

Gerador de senhas aleatórias feito em Python.

## Funcionalidades

- Gerar senhas com letras, números e símbolos
- Escolher o tamanho da senha
- Gerar múltiplas senhas de uma vez
- Garantia de pelo menos 1 maiúscula, 1 número e 1 símbolo
- Avaliação de força da senha (Fraca, Média ou Forte)
- Copia a senha automaticamente para o clipboard
- Menu interativo com interface limpa

## Como usar

1. Clone o repositório
2. Instale a dependência: `pip install pyperclip`
3. Execute: `python gerador.py`

## Tecnologias

- Python 3.14
- random
- string
- pyperclip
- os

# 🔐 Gerador de Senhas

Gerador de senhas seguras desenvolvido em Python, com foco em segurança, usabilidade e automação.

---

## ✨ Funcionalidades

- Geração de senhas aleatórias
- Letras maiúsculas e minúsculas
- Números e símbolos
- Escolha personalizada do tamanho
- Geração de múltiplas senhas
- Garantia de:
  - 1 letra maiúscula
  - 1 número
  - 1 símbolo
- Avaliação de força da senha
- Cópia automática para clipboard
- Interface interativa no terminal

---

## 🛠 Tecnologias Utilizadas

- Python 3.14
- `random`
- `string`
- `pyperclip`
- `os`

## 🚀 Como Executar

Clone o repositório:

```bash
git clone https://github.com/pietro-sys/gerador-de-senhas.git
```

Entre na pasta do projeto:

```bash
cd gerador-de-senhas
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o programa:

```bash
python gerador.py
```
